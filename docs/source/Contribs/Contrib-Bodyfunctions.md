# Script 範例

Griatch 的貢獻，2012 年

用於測試的範例 script。這新增了一個簡單的計時器，其中包含您的
人物會不定期地進行一些小小的口頭觀察。

要測試，請使用（在遊戲中）

    > script me = contrib.tutorials.bodyfunctions.BodyFunctions

(notes)=
## 筆記

使用 `scripts me` 檢視您身上執行的 script。請注意，即使
計時器滴答到 0，你不會在每次滴答時看到迴聲（這是
是否在刻度上給出迴聲是隨機的）。
