# TutorialMirror

Griatch 的貢獻，2017 年

一個簡單的鏡子物件進行實驗。它會對被注視做出反應。

- 回顯所看到的物體的描述
- 回顯傳送到其.msg 的任何內容 - 到
傳送者，如果給出的話，否則到映象的位置。

(installation)=
## 安裝

建立映象

    create/drop mirror:contrib.tutorials.mirror.TutorialMirror

然後看看它。
