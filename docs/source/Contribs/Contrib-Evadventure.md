使用 Evennia 的完整範例 MUD。這是最終的結果是什麼
如果您遵循[入門教學的第 3 部分](../Howtos/Beginner-Tutorial/Part3/Beginner-Tutorial-Part3-Overview.md)，則可以實現。
建議您按照教學一步步編寫自己的
程式碼。但如果您願意，您也可以拆開或使用它作為起點
為了你自己的遊戲。

(features)=
## 特徵

- 使用MUD-版本的[Knave](https://rpggeek.com/rpg/50827/knave)老式
本·米爾頓 (Ben Milton) 的奇幻規則集（無階級且總體與早期相容）
  D&D 版），根據知識共享署名發布（所有用途，
  包括商業廣告都是允許的
  只要給出歸屬）。
- 使用可編輯的角色表建立角色
- 武器、效果、治療和休息
- 兩種可供選擇的戰鬥系統（回合製和抽搐式）
- 魔法（三個咒語）
- NPC/帶有簡單AI的生物。
- 簡單的任務系統。
- 小遊戲世界。
- 使用最佳 Evennia 實踐進行編碼，並進行單元測試。
