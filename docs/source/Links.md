# 連結

*這裡整理了一些對 Evennia 使用者與開發者可能有幫助的資源。*

## Evennia 官方資源

- [evennia.com](https://www.evennia.com) - Evennia 的主入口網站，幾乎所有官方資源都能從這裡找到。
- [Evennia development blog](https://www.evennia.com/devblog/index.html) - 主要開發者的隨筆與開發觀察。
- [Evennia on GitHub](https://github.com/evennia/evennia) - 下載程式碼並閱讀文件。
- [Evennia on PyPi](https://pypi.python.org/pypi/evennia)
- [Evennia on Open Hub](https://www.openhub.net/p/6906)

## Evennia 社群

- [Evennia Game Index](http://games.evennia.com/) - 自動整理的 Evennia 遊戲列表。
- [Evennia official Discord channel](https://discord.gg/AJJpcRUhtF) - 聊天、提問並取得協助！
- [Evennia official forums](https://github.com/evennia/evennia/discussions) on Github Discussions.
- [Evennia subreddit](https://www.reddit.com/r/Evennia/) - 互動速度稍慢一些，但依然活躍。

## 第三方 Evennia 資源

- [Search Evennia docs with chatGPT](https://chat.openai.com/g/g-Qu7AJQEbZ-evennia-guide) - 載入 Evennia 文件的自訂 chatGPT（截至 2023 年 12 月）。
- [Evennia-related repos on github](https://github.com/search?p=1&q=evennia)
- [Evennia minimud](https://github.com/InspectorCaracal/evennia-minimud) - 只使用 contrib 建立的極簡 Evennia 遊戲，基於 Evennia 2.0.0 程式碼，很適合拿來當起點！
- [EvscapeRoom](https://github.com/Griatch/evscaperoom) - 以 Evennia 製作的多人密室逃脫完整程式碼（引擎本身是一個 [Evennia contrib](Contribs/Contrib-Evscaperoom.md)）。
- [Arx sources](https://github.com/Arx-Game/arxcode) - 很受歡迎的 [Arx](https://play.arxmush.org/) Evennia 遊戲的開源程式碼。[這裡有安裝說明](https://www.evennia.com/docs/1.0-dev/Howtos/Arxcode-Installation.html)。它使用的是相對較舊版本的 Evennia。
- [Encarnia sources](https://github.com/whitehorse-io/encarnia) - 一個針對較舊 Evennia 版本開源的 game dir，包含種族、戰鬥等內容。[摘要在這裡](https://www.reddit.com/r/MUD/comments/6z6s3j/encarnia_an_evennia_python_mud_code_base_with/)。
- [The dark net/March Hare MUD](https://github.com/thedarknet/evennia) - 來自 2019 年巴黎 [DEF CON 27](https://www.defcon.org/html/defcon-27/dc-27-index.html) 駭客大會的 Evennia game dir，使用 batchcode 建出客製化的 _Hackers_ 風格網路空間區域與解謎內容，[活動中就是用它](https://dcdark.net/home#)。

### 其他

- [Evennia for MUSHers](https://musoapbox.net/topic/1150/evennia-for-mushers) - 一篇寫給熟悉 MUSH 工作方式的玩家與開發者的 Evennia 介紹文章。
- *[Language Understanding for Text games using Deep reinforcement learning](http://news.mit.edu/2015/learning-language-playing-computer-games-0924#_msocom_1)* ([PDF](https://people.csail.mit.edu/karthikn/pdfs/mud-play15.pdf)) - MIT 使用 Evennia 訓練 AI 的研究論文。

## 一般 MU* 資源

### MU* 工具

- [ROM area reader](https://github.com/ctoth/area_reader) - 可把 ROM area 檔轉成 Python 物件的 parser。

### 參考資料

- [writing games](https://writing-games.com) - 蒐集文字遊戲相關文章與部落格的策展站，裡面也有不少 MU 內容。
- [Imaginary Realities unofficial archive](http://tharsis-gate.org/articles/imaginary.html) - 一份關於遊戲與 MUD 設計的電子雜誌存檔，其中包含多篇和 Evennia 有關的文章。
- [Lost Library of MOO](https://www.hayseed.net/MOO/) - 與 mudding 相關的學術文章典藏，特別偏向 moo。
- [Mud Client/Server Interaction](http://cryosphere.net/mud-protocol.html) - 介紹經典 MUD telnet protocols 的頁面。
- [Mud-dev wiki](http://mud-dev.wikidot.com/) - 一個（非常）慢慢成長中的 MUD 製作資源站。
- [Mud Tech's fun/cool but ...](https://gc-taylor.com/blog/2013/01/08/mud-tech-funcool-dont-forget-ship-damned-thing/) - Greg Taylor 提供了不少很實用的 mud 設計建議。
- [Nick Gammon's hints thread](http://www.gammon.com.au/forum/bbshowpost.php?bbsubject_id=5959) - 整理出一份非常值得在開始做新 MUD 前先思考的清單。
- [Raph Koster's laws of game design](https://www.raphkoster.com/games/laws-of-online-world-design/the-laws-of-online-world-design/) - 一套很能引發思考的準則，適合拿來設計虛擬多人世界（Raph 也以 *Ultima Online* 等作品聞名）。

### 書籍與閱讀

- Richard Bartle *Designing Virtual Worlds*
  ([amazon page](https://www.amazon.com/Designing-Virtual-Worlds-Richard-Bartle/dp/0131018167)) -
  這本書是設計任何持久型遊戲世界時的經典必讀，由原始 *MUD* 的共同創作者撰寫。雖然出版於 2003 年，但直到今天依然很有參考價值，涵蓋的內容非常全面。

  後來當 *Designing Virtual Worlds* 的權利回到 Richard Bartle 手上後，他也透過自己的網站免費提供這本書的 PDF（[Designing Virtual Worlds](https://mud.co.uk/dvw/)）。PDF 直接連結在[這裡](https://mud.co.uk/richard/DesigningVirtualWorlds.pdf)。
- David M. Beazley *Python Essential Reference (4th ed)* ([amazon page](https://www.amazon.com/Python-Essential-Reference-David-Beazley/dp/0672329786/)) - 我們很推薦的 Python 書籍；它不只高效率整理了這門語言，也是一份對較有經驗 Python 開發者相當實用的標準函式庫參考書。
- Luciano Ramalho, *Fluent Python* ([O'Reilly page](https://shop.oreilly.com/product/0636920032519.do)) - 非常適合已有經驗、想讓自己程式碼再進一步的 Python 開發者。內容很紮實，即使是老手也能從中獲得不少收穫。
- Richard Cantillon *An Essay on Economic Theory* ([free pdf](https://mises.org/books/essay_on_economic_theory_cantillon.pdf)) - 這是 *Essai sur la Nature du Commerce en Général* 很優秀的一份英文譯本，也是現代經濟理論的奠基文本之一。雖然原作寫於 1730 年，但譯文附有註釋，而且對現代讀者來說其實不難讀。如果你打算為遊戲實作一套合理的經濟系統，非常值得一讀。
